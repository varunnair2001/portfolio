VoloFin Portal Redesign

Introduction

VoloFin is a fintech platform that uses blockchain to offer easy invoice factoring for SMEs. It ensures transparency, integrity, and visibility.


The Previous Version

The internal platform simplifies the process of checking customer information for different teams, enabling them to easily verify them. Once approved, the client portal allows for secure and efficient invoice transactions, all in four simple stages:

Customer Onboarding

Credit Underwriting

Submitting Invoices

Disbursement of Funds

Challenges with the Previous Version


Previous Version Demo

Tedious Onboarding

Primitive User Interface

Flawed Workflow

No Process Automation

Project Brief

1. To ideate an entirely new workflow that fixes various shortcomings of the current platform structure

2. Future-proof the system such that it can easily adopt additional functionalities in the next five years

3. Introducing an entirely new Sales CRM that incorporates into the existing process flow which includes - Lead Generation, Offer Issuance, Client Onboarding

4. Introducing information repositories that can store credit underwriting for various clients for multiple financial years - Client Revisions, Renewals, and Re-Applications

5. Integrating multiple APIs for various steps of the process flow to automate majority of the tasks

6. Redesigning the entire UI with a few objectives - changing the primary color, improving information architecture, introducing module roles (makers & checkers)

7. Introducing Channel Partnership Module, Partner Module, and Investor Module enabling a client referral system & investor profile monitoring

8. Adding a real time notification system for all users
